Wikispeedia

The prettiest Wikipedia game on the internet.

The Wikipedia game is an old standby for bored teens, but it's rarely done right. For our final project for CS 32 at Brown, McKenna Cisler, Rohan Jha, Sean Hastings and I decided to perfect this age-old game. I worked on the user experiene, including lobby experience, sitewide design, and gaphic design. I also set up hosting so anyone can play online.

Want to play the game yourself? You can play here. Since it is hosted on Heroku for free, please give the app a few seconds to start up. View our project git here.

  • Date: March - May 2017
  • Categories: Hosting, UI/UX, Frontend, Web Development, Graphic Design

EQUiSat

Making space accessible.

At Brown Space Engineers, we are dedicated to making space more accessible. Following our directions, anyone can make a satellite for only $5,000. Launching in Q2 2018, EQUiSat will be the first satellite to test LiFePO batteries in space and its four LEDs will be visible by the naked eye. I have worked on organizing satellite operations using FreeRTOS and optimizing data transmission, especially error packaging.

For more information, you can visit the team website here. View our project git here.

  • Date: September 2016 - Present
  • Categories: Systems, Avionics

The Social Playlist

Make the party's playlist communal.

Ever wanted to play music at a party but allow your friends to pick some of the music? The Social Playlist attempts to solve that problem by allowing a host to set up a room which visitors can join. Visitors can suggest songs (which are automatically queued on YouTube) which the host can rearrange or reject. We also attempted to use Microsoft's Emotion API to suggest a playlist based on the mood detected by a device's webcam. This project was created at Hack@Brown 2017. I made all user-facing components and attempted to integrate Microsoft's Emotion API.

For installation directions and more information, view our project git here.

  • Date: February 2017
  • Categories: UI/UX, Frontend, Web Development
