this-site

Work in progress

Since mid 2016, I've been continuously redesigning jleiken.com, this site. In this most recent iteration, I decided to mimic the experience of using a terminal. Nearly every semester in college I found myself writing a REPL, so I have very strong ideas about what I like and what I don't. Drawing on that experience, combined with my modern Javascript experience at Amazon, I decided to write this website in pure ES, no jQuery involved. The static version, for mobile and visitors unfamiliar with terminals, serves as a template for the REPL version. This means that the Javascript can pull from the HTML - there are very few strings that contain HTML code.

The site is hosted on Github Pages and uses Jekyll for Sass. If you're interested in how I wrote this site, you can view the source code on Github. The REPL is defined in js/repl.js and run from js/index.js.
